Управление Штрих-Кодами

Для того чтобы начать пользоваться картами скидок, давайте сначала посмотрим на управление штрих-кодами для того, чтобы распечатать наши правильные карты для скидок.

Например, я хочу иметь скидку для товара со следующим штрих-кодом.

Откройте:menuselection:[UNKNOWN NODE title_reference]. Здесь вы можете увидеть, что для того чтобы установить скидку, ваш штрих-код должен начинаться с числа [UNKNOWN NODE title_reference], а затем вам нужно добавить процент, который вы хотите установить для товара.

Сканирование товаров

Если вы вернетесь к инфо-панели и начнете новую смену

Вы должны отсканировать:

  1. товар

  2. карту скидок

Когда товар отсканирован, он появляется в чеке

Тогда при сканировании карты скидок, 50% скидка будем применена к товару.

Таким образом вы можете использовать карты скидок для товаров в Odoo.
