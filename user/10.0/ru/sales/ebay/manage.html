Объявление с одним вариантом товара

В запросе на размещения продукта, вам необходимо проверить поле use eBay в бланке продукта. Станет доступна вкладка eBay.

Когда опция Использовать количество товара (Use Stock Quantity) установлена, количество запросов с eBay будет отображаться в Odoo как Прогнозируемое количество (Forecast Quantity).

Шаблон описания (Description Template) позволяет использовать шаблоны для ваших объявлений. Шаблон по умолчанию использует только поле eBay Description**(описание) данного продукта. Вы можете использовать html при редактировании **Шаблон описания (Description Template) и eBay Description.

Чтобы использовать изображения в своем объявлении, вам необходимо добавить их как Вложения (Attachments) в шаблон продукта.

Объявления с несколькими вариантами товара

Когда опция Использовать eBay (use eBay) в бланке продукта с вариантами, и с фиксированной ценой (Fixed Price) как Тип размещения объявления (Listing Type), форма eBay будет несколько отличатся. Из множества вариантов вы можете выбрать, какой вариант будет указан на eBay, а также установить цену и количество для каждого варианта.

Объявление с характеристиками товара

Чтобы добавить характеристики товара, вы должны создать атрибут продукта с одним значением на вкладке Варианты (Variants) в карточке продукта.

Штрих-коды продуктов

В большинстве категорий eBay требует штрих-коды продуктов, такие как EAN, UPC, Brand или MPN. Штрих-коды EAN и UPC редактируются в поле Штрих-код (Barcode) в варианте продукта. Если поле Штрих-код (Barcode) пусто или значение недопустимо, значения EAN и UPC будут установлены как «Не применяется (Does not apply)», как это рекомендовано eBay. Значения Brand и MPN работают как характеристики товара и должны быть определены на вкладке Варианты в карточке продукта. Если эти значения не установлены, для объявления eBay будет использоваться значение - «Не применяется (Does not apply)».
